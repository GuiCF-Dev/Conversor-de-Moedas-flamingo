# 💱 Conversor de Moedas com API

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de um **Conversor de Moedas** utilizando **JavaScript** e integração com uma **API de câmbio em tempo real**.

A aplicação permite ao usuário selecionar moedas diferentes e realizar conversões automaticamente com base nas cotações atualizadas fornecidas pela API, proporcionando uma experiência mais próxima de aplicações utilizadas no mercado.

---

## 🎯 Objetivos

* Consumir dados de uma API externa.
* Trabalhar com requisições HTTP utilizando JavaScript.
* Manipular dados em formato JSON.
* Atualizar informações dinamicamente na interface.
* Desenvolver aplicações interativas utilizando JavaScript.
* Compreender o fluxo de comunicação entre Front-End e APIs.
